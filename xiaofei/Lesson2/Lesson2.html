1.变量

变量是用于存储信息的容器

1.1 变量名最好是用字母开头

1.2 要取一些有意义的名字

1.3 区分大小写

2.数据类型

2.1基本类型：字符串("程序员小飞"),数字(19),布尔(true/false),对空(Null),未定义(Undefined),Symbol(独一无二的值)

2.2引用类型: 对象、数组、函数

var x;

var x = 5;

var x = 3.14;

var x = 123e5;

var x = "程序员小飞";

var x = true;

var x = false;

var cars = new Array();
cars[0] = "Saab";
cars[1] = "Volvo";
cars[2] = "BMW";

或者这样子初始化数值也行：
var cars = new Array("Saab","Volvo","BMW"); 或者还有：
var cars = ["Saab","Volvo","BMW"];

var person = {"name":"程序员小飞","age":28};

3.对象

对象是拥有属性和方法的数据

var person = {"name":"程序员小飞","age":28,"eyeColor":"blue",fullname:function(){alert("程序员小飞");}};

person.name 或者 person["name"]

5.变量的生存期

5.1 在函数内部声明的变量,只能在函数内部访问它,出了函数就销毁了;

5.2 在函数外部声明的变量,可以全局访问;

5.3 如果函数内部的变量没有带var,表示它是全局变量,可以在函数外访问;
